#  Gestor de Productos con JavaScript y API REST

Aplicación web desarrollada con:

- HTML
- CSS
- JavaScript

Permite administrar productos mediante operaciones CRUD:

 Crear productos  
 Leer productos  
 Editar productos  
 Eliminar productos  

Además, sincroniza información con una API REST usando `fetch()` y guarda datos localmente mediante `localStorage`.

---

#  Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
- DOM
- LocalStorage
- Fetch API
- API REST

---

#  Funcionalidades

 Agregar productos  
 Editar productos  
 Eliminar productos  
 Persistencia de datos con `localStorage`  
 Sincronización con API REST  
 Modal de edición  
 Validación de formularios  
 Contador dinámico de productos  
 Manipulación dinámica del DOM
